## 运行

小可目前支持以下平台：
- [X] QQ
  - [X] 官方机器人
  - [X] 第三方协议
- [X] Discord
- [X] Telegram
- [X] KOOK（开黑啦）
- [X] Matrix

### 自搭建

你可以参考[此处](https://bot.teahouse.team/wiki/%E8%87%AA%E6%90%AD%E5%BB%BA)（[镜像](https://github.com/Teahouse-Studios/akari-bot/wiki/%E8%87%AA%E6%90%AD%E5%BB%BA%E6%8C%87%E5%8D%97)）来尝试搭建。
