## 介绍

**小可**是可扩展的多功能、多语言、多平台支持的异步聊天机器人。当前形象由 [OasisAkari](https://github.com/OasisAkari) 绘制。

目前小可正在进行快速开发中，特性可能会随时发生变化，请留意更新。
