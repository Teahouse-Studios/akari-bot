## 介绍

**小可好耶机**是基于多个框架开发的可扩展、多平台机器人。图标来源于 [P站](https://www.pixiv.net/artworks/98830047)。

目前小可正在进行快速开发中，特性可能会随时发生变化，请留意更新。

## 平台

小可目前支持以下平台：

-   QQ（官方/第三方）
-   Discord
-   Telegram
